# Generador de Post (FrontEnd)

## Funcionalidad

Permite registrarse e iniciar sesión.

Una vez ingresando en la aplicación podras generar posts con imagenes de tu mismo dispositivo y podras agregarles un titulo y una descripción.

### Tecnologias utilizadas
HTML, CSS, JavaScript, React, React Router Dom, React Icons y Bootstrap.
